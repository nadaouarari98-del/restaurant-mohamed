# MANAGISH مناقيش | Authentic Lebanese Restaurant

A static website for **MANAGISH**, an authentic Lebanese restaurant offering fresh manakish, fine dining, elegant venue rentals, and premium catering services.

## Pages

- **Home** (`index.html`) – Landing page with hero section and highlights
- **About** (`about.html`) – Story and philosophy of the restaurant
- **Menu** (`menu.html`) – Full menu with dishes and descriptions
- **Venue** (`venue.html`) – Venue rental information
- **Catering** (`catering.html`) – Catering services details
- **Contact** (`contact.html`) – Contact form and location info

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Netlify CMS](https://www.netlifycms.org/) for content management (`admin/`)
- Hosted on [Netlify](https://www.netlify.com/)

## Development

This is a static site — open any `.html` file directly in your browser, or use a local development server:

```bash
npx serve .
```

## Content Management

Content is managed via Netlify CMS. Access the admin panel at `/admin` once deployed.

## Deployment

The site is configured for Netlify deployment via `netlify.toml`. Push to the main branch to trigger a deployment.
